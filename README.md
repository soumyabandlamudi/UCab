# UCab – Cab Booking System

## Project Overview
UCab is a full-stack Cab Booking System developed using the MERN Stack. The application enables users to book cabs online, manage bookings, and allows administrators to manage users, cabs, and bookings through an admin dashboard.

## Features
- User Registration and Login
- Secure Authentication
- Browse Available Cabs
- Book a Cab
- View Booking History
- Cancel Bookings
- Admin Dashboard
- Manage Users
- Manage Cabs
- Manage Bookings
- Responsive User Interface

## Technologies Used

### Frontend
- React.js
- HTML5
- CSS3
- JavaScript

### Backend
